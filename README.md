# Matrix Calculator

A responsive matrix calculator built with:
- Next.js App Router
- TypeScript
- Material UI
- Responsive Design

## 🖼️ Screenshot
> ![website-img](/matrix-calculator/public/images/website-image.png)

## 🚀 Features
- Generate Matrix A (i + j)
- Generate Matrix B (i × j)
- Add matrices and view result
- Responsive layout
- Input validation

## 🔧 Setup
1. Clone repo
2. `npm install`
3. `npm run dev`
4. Open `http://localhost:3000`
